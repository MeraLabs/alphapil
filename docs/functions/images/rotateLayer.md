# rotateLayer

Alias for [$rotate](rotate.md). Rotates the main canvas or active layer.

## Syntax

```bash
$rotateLayer[angle;expand]
```

## Parameters

| Parameter | Type | Description | Default |
| :--- | :--- | :--- | :--- |
| **angle** | `float` | Angle of rotation in degrees counter-clockwise. | Required |
| **expand** | `bool` | If `true`, expands the canvas size to fully fit the rotated image. | `false` |

See the [rotate](rotate.md) page for full details and examples.
