# toUpper

Converts a string to uppercase.

## Syntax

```bash
$toUpper[text]
```

## Parameters

| Parameter | Type | Description | Default |
| :--- | :--- | :--- | :--- |
| **text** | `string` | The text string to convert. | Required |

## Example

```bash
# Store an uppercase string in a variable
$setVar[title;$toUpper[hello world]] # Stores "HELLO WORLD"
```
